Getting Started

APIToolz is a back end development and web service (API) GUI tool that is to create and manage all your API, CURD form and more features without any coding.

API Toolz is a powerful automation tool that can create a full set of PHP quickly from MySQL. You can instantly create web sites that allow users to view, edit, search, add and delete records on the web. API Toolz is designed for high flexibility, numerous options enable you to generate PHP applications that best suits your needs. The generated codes are clean, straightforward and easy-to-customize. API Toolz can save you tons of time and is suitable for both beginners and experienced develpers alike

Server & PHP Requirment

To make Sximo Builder running on your , you need to make sure your server match with minimum requirment

  1. Apache, nginx, or another compatible web server.
  2. PHP >= 5.5.9 and Above
  3. MySQL Database Server
  4. OpenSSL PHP Extension
  5. Mbstring PHP Extension
  6. Tokenizer PHP Extension
  7. Support for Module Re_Write Enable
  8. Support for PHP_CURL Module Enable

Installation

Preparing Installation

  1. Download ZIP package and unzip it.
  2. In unzipped folder you will find folder directory like bellow.
    • apitoolz
    • database
    • documentation
  3. Create app directory on your web directory server
  4. Simply put all files inside directory apitoolz to your app folder.

Create new database installation

You need to create new database , you can use phpmyadmin or any mysql databse tools such maestro , sqlyog etc
---
After new database created , now import database/dev.apitoolz.sql file into your new database

Configuration Installation

1. Open apitoolz/.env file using your favorite php editor

2. Change with your application info... 

		APP_NAME='API Toolz'
		APP_DESC='Dont give up your time on coding, API Toolz will coding for you'
		APP_METAKEY='API Toolz'
		APP_METADESC='Dont give up your time on coding, API Toolz will coding for you'
		APP_COMNAME='M S D T Company Limited'
		APP_EMAIL='info@msdt.com.mm'
		APP_TIME='Asia/Rangoon'
		APP_URL='https://apitoolz.com'

3. Change Database Info

Open apitoolz/.env file using your favorite php editor 

		DB_DATABASE=' YOUR CREATED NEW DB NAME '
		DB_USERNAME=' YOUR MYSQL USERNAME '
		DB_PASSWORD=' YOUR MYSQL PASSWORD '

4. Access your application

  1. Open terminal, and go to your application directory
    Run php artinsan serve
  2. Now access to your application via your favorite browser
    http://127.0.0.1:8000
  3. Default Login :
    Email Address : superadmin@mail.com
    Password : superadmin
  4. Awesome your application is ready now !

5. Files Folder permission

please make sure to set permission to 0775 at following file and folder on linux/unix server

  1. /app/Http/Controllers/
  2. /app/Http/Models/
  3. /resources/views/
  4. /storage/
  5. /routes/

6. Troubleshoot

  1. Page not found after click any link

    You need to make sure enabling module apache re_write module to make sure htaccess working

Settings for Application

After your successfull install your application , now you need to setting your app.
Log to your application and go to :

Go to : Actions -> Settings menu

General Settings

Security Settins

Manage all things these are login redirect, default register as, activation and block ip if you want.

SWIFT MAIL

If you want to use SWIFT mail as your default mail server , follow this step

  • Go to : Actions -> Settings -> Security -> SWIFT Mail ( Required Configuration )
  • Select Mail System ( SWIFT Mail ) and click "Required Configuration"
  • Change following mail config match your server info 
    			MAIL_DRIVER=smtp
			MAIL_HOST=smtp.gmail.com
			MAIL_PORT=587
			MAIL_USERNAME=apitoolz.automail@gmail.com
			MAIL_PASSWORD=youremailpassword
			MAIL_ENCRYPTION=tls
			MAIL_FROM_ADDRESS=apitoolz.automail@gmail.com
			MAIL_FROM_NAME=APIToolz

  • If you have done, click on "Save changes"

Database Management

What's database management ? Database management is base required for data storage of your application and to create new table/view that used when created new model of applicaton. And also can import from existing database structure of your.

To create database table

Go to : Actions -> Database Tables -> New Table menu

To create database table

Go to : Actions -> Database Tables -> New View menu

To import from existing database table structure

Go to : Actions -> Database Tables -> Import Table menu

Model

What's Model ? A model is a part of a app.A model contain one or several routines that we call CRUDSD ( Create, Read, Update, Delete, Search and Download ).
By default at the first time you create model , system will generate thoose routines.

Create New Model

Go to : Actions -> View All Models > Create Model

In API Toolz , we have added 2 option of Model.

  • Active if you checked this option, this model will active to production.
  • Create report view onley if you checked this option, this model will only create listing of database table.

Edit to Current Model

Go to : Actions -> View All Models > Actions of each Model > Configuration

Edit to Model Info

Edit to SQL

Go to : Actions -> View All Models > Actions of each Model > Configuration > SQL tab

Settings for Table (Listing of each Model)

Go to : Actions -> View All Models > Actions of each Model > Configuration > Table tab

Settings for Form (Entry Form/Sub Form/Layout Design of each Model)

Go to : Actions -> View All Models > Actions of each Model > Configuration > Form tab

Join to Relation with other Model

Go to : Actions -> View All Models > Actions of each Model > Configuration > Relation tab

Settings for Email Notification

Go to : Actions -> View All Models > Actions of each Model > Configuration > Notification tab
